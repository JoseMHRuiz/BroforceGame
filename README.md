# BroforceGame!!

Hi! This is my first project

Made with HTML5, CSS, VJScript, Canvas, GSAP, Howler.js

# Controls

First pick a player, just different sprite.
For shot pick your mouse and push **SPACE**, if you want to throw some grenades push **C**

Have fun!
